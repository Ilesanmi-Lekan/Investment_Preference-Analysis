# Investment Preference Analysis

## Project Overview
This project analyzes customers investment preferences to understand their preferred investment options and patterns using Excel.

## Dataset
The dataset contains customer investment choices and related attributes.
It was analyzed and cleaned using Excel.

## Tools Used 
- Microsft Excel

## Analysis Process 
- Data cleaning and formatting
- Pivot tables and summary statistics
- Charts and visual analysis

## Key Insights
- Most customers invested in mutual funds
- Established adulthood show higher investment activity
- Returns is the key investment decision drivers

## Recommendation
- Target low risk products to conservative investors
- Design age specific investment plans
